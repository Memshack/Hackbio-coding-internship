# Hackbio-coding-internship
This repository contains all materials, scripts, and analyses completed during the HackBio Internship, where training focuses on applying computational tools to biological data across multiple domains. The work integrates both R and Python for data handling, statistical analysis, and reproducible research workflows in bioinformatics. The internship covers key areas including genomic epidemiology, transcriptomics, genomics, chemoinformatics, and metagenomics. Each section documents practical exercises and projects that apply core concepts such as sequence analysis, gene expression profiling, molecular data interpretation, and structure-based chemical analysis.
